# 刑事证据法定类别

依据《刑事诉讼法》第五十条，刑事证据包括以下类别。使用本文件时，应另行核验现行法律文本。

## 八类证据

| 类别 | 典型材料 | 质证重点 |
|---|---|---|
| 物证 | 作案工具、赃物、毒品、样本、痕迹载体 | 来源、提取、封存、保管、移送、同一性、污染风险 |
| 书证 | 合同、账册、银行流水、户籍、报案材料、扣押清单 | 原件/复制件、形成时间、出具主体、签章、关联性 |
| 证人证言 | 证人询问笔录、情况说明 | 感知条件、利害关系、前后矛盾、是否出庭、诱导可能 |
| 被害人陈述 | 报案笔录、陈述笔录、辨认说明 | 与客观证据是否印证、动机、损失依据、陈述变化 |
| 犯罪嫌疑人/被告人供述和辩解 | 讯问笔录、亲笔供词、同步录音录像 | 讯问合法性、同步录音录像、供述变化、补强证据、翻供理由 |
| 鉴定意见 | DNA、毒品、价格、伤情、精神病、痕迹、电子数据鉴定 | 机构/鉴定人资质、检材来源、方法、过程、意见形式、重新鉴定 |
| 勘验、检查、辨认、侦查实验等笔录 | 现场勘验、辨认、搜查、检查、侦查实验 | 见证人、程序、签名、照片/录像对应、辨认混杂和暗示 |
| 视听资料、电子数据 | 监控、录音录像、聊天记录、手机数据、服务器日志 | 提取固定、原始载体、完整性校验、哈希、时间戳、关联账号 |

## 扩展问题

- 技术侦查/调查证据：审查审批手续、措施种类、转化材料、是否超范围使用。
- 行政执法证据转化：审查行政机关取证主体、程序、移送链条和刑事诉讼中的可采性。
- 监察证据转化：审查形成主体、同步录音录像、移送手续、与刑事证据规则的衔接。

## 输出要求

所有证据分类都必须落到具体定位信息：卷名、原始文件名或路径、PDF 页序号、卷宗页码、OCR 状态、原件核验状态。
