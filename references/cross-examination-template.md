# 质证意见模板

质证意见用于审查起诉阶段辩护意见和庭审质证准备。每项证据必须带定位信息，方便庭审快速查找原文。

```markdown
# 质证意见 - [案件名称]

**适用场景：** 审查起诉阶段辩护意见 / 庭审质证准备
**说明：** 本意见基于现有卷宗材料和 OCR 索引形成，关键事实已/未完成原件核验情况见各项。

## 一、总体质证意见

[概述证据体系是否完整、是否存在合理怀疑、主要程序问题、主要证据缺口。]

## 二、逐项质证意见

| PDF页序号 | |

**真实性意见：**

[原件、来源、同一性、完整性、矛盾点。]

**合法性意见：**

**关联性意见：**
```

## 质证策略词汇

- 排除：证据取得严重违法或依法不得作为定案根据。
- 削弱：证据可进入审查，但证明力不足。
- 限制证明目的：证据只能证明形式事实、来源事实或背景事实，不能证明控方核心待证事实。
- 要求补强：言词证据、复制件、电子数据、鉴定意见等需要其他证据印证。
- 保留异议：暂未能核验原件、同步录音录像、原始载体或完整链条。
